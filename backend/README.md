# School Management Node.js API

## Tech Stack
- Node.js (ES Modules)
- Express.js
- MySQL2
- Zod (Validation)

## Setup
1. Create a MySQL DB and run \schema.sql\ to instantiate the \schools\ table.
2. Create a \.env\ file in the root based on \.env.example\ or just provide:
   - \DB_HOST\, \DB_USER\, \DB_PASSWORD\, \DB_NAME\`n3. Run \
pm start\ or \
ode src/index.js\.
